## 🔧 Особливості CSS:

### Компоненти:

1. **Заголовок форуму** (`.forum-header`)
   - Синя межа знизу
   - Великий заголовок
   - Підзаголовок-описання

2. **Секція розділу** (`.forum-section-header`)
   - Темний фон із білим текстом
   - Іконки для кожної секції

3. **Картка категорії** (`.category-box`)
   - Біле тло
   - Тінь при наведенні
   - Статистика справа

4. **Таблиця тем** (`.threads-table`)
   - Переривчастий рядок при наведенні
   - Центрована статистика
   - Лінійні посилання синього кольору

5. **Кнопки** 
   - `.btn-forum-primary` — основні дії (синій)
   - `.btn-forum-secondary` — додаткові дії (сірий)

6. **Форма** (`.form-control`)
   - Ясна межа
   - Фокус-стани
   - Текстареї з мінімальною висотою
